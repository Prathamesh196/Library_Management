# Library Management System (C++)

A console-based Library Management System developed using C++ to demonstrate strong fundamentals in Object-Oriented Programming, STL, and problem solving. The application simulates basic library operations through a menu-driven interface and follows a clean, modular design suitable for core software engineering roles.

## Features
- Add new books to the library
- Display all available books
- Issue books to users
- Return issued books
- Menu-driven console interaction
- Basic input validation

## Technologies Used
- Programming Language: C++
- Core Concepts: Object-Oriented Programming, Data Structures
- STL: vector
- Tools: VS Code, g++ compiler
- Version Control: Git and GitHub

## Design Overview
The project is designed using object-oriented principles. The Book class represents individual books and stores details such as book ID, title, author, and issue status. The Library class manages a collection of books using an STL vector and handles operations like adding, issuing, returning, and displaying books. The main program provides a menu-driven interface and controls the application flow based on user input.

## How to Run
1. Clone the repository  
2. Navigate to the project directory  
3. Compile the program using  
   g++ main.cpp Book.cpp Library.cpp -o library  
4. Run the application using  
   ./library
